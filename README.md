🌍 *Read this in [English](#english) | Léelo en [Español](#español)*

## <a name="english"></a> 🇺🇸 English

A classic implementation of the beloved Snake game, built entirely for the Windows console.

### 📖 About This Project

I originally wrote this game back in 2022 during my very first year studying Software Engineering. I've decided to pull it out of the archives and upload it to GitHub to immortalize where my software engineering journey began. 

While the codebase might be a bit rough around the edges compared to my current standards, it serves as a fun time capsule. If you are just starting out with C++, you might find this repository helpful to see how fundamental concepts (like basic game loops, console I/O, and simple logic) come together. Feel free to explore the code, modify it, and learn from it!

### ✨ Features

* Classic Snake mechanics (movement, eating, growing).
* Runs directly in the console.
* Written in pure C++.
* A great, straightforward reference for C++ beginners.
